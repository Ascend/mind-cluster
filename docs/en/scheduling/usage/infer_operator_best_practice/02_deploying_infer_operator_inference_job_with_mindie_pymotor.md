# Deploying Infer Operator Inference Jobs Based on MindIE PyMotor

MindIE PyMotor is an inference cluster management framework developed by Ascend, which supports the generation and deployment of Infer Operator inference Jobs. For the detailed deployment process of submitting inference Jobs based on MindIE PyMotor, see [CRD Deployment Design](https://gitcode.com/Ascend/MindIE-PyMotor/blob/master/docs/zh/design/crd_deployment.md).
