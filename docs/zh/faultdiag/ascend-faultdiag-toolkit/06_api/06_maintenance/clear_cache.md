# clear_cache

## 命令功能

清理缓存。

- 清理工具运行过程中产生的缓存文件。
- 建议在执行新诊断任务前执行，避免上次诊断结果干扰本次诊断。
- 若清理未生效，请使用管理员模式打开工具。

## 命令格式

| 命令格式 | 描述 |
|---------|------|
| `clear_cache` | 清理链路诊断工具缓存。 |
| `clear_cache ?` | 查看详情。 |

## 参数说明

无参数。

## 输出说明

- 成功：`清理完成`
- 失败：`清理{collect_cache}异常：{err}`

## 示例

非交互式方式：

```bash
ascend-fd-tk clear_cache
清理完成
```

交互式方式：

```bash
ascend-fd-tk
>>> clear_cache
清理完成
```
