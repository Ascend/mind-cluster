# clear

## 命令功能

清空当前终端屏幕的显示内容，便于查看后续输出。该命令仅影响屏幕显示，不会清理工具内部状态或缓存数据。

## 命令格式

| 命令格式 | 描述 |
|---------|------|
| `clear` | 清屏 |
| `clear ?` | 查看详情 |

## 参数说明

无业务参数，`?` 为内置帮助标识，用于查看命令用法。

## 输出说明

执行后清空当前终端屏幕显示内容（不影响工具内部状态）。

## 示例

非交互式方式（展示命令与回显）：

```bash
ascend-fd-tk clear
```

交互式方式（展示命令与回显）：

```bash
ascend-fd-tk
>>> clear
```
