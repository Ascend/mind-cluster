# exit

## 命令功能

退出诊断工具交互式程序。执行后工具进程结束。该命令仅在交互式模式下有意义。

## 命令格式

| 命令格式 | 描述 |
|---------|------|
| `exit` | 退出程序 |
| `exit ?` | 查看详情 |

## 参数说明

无业务参数，`?` 为内置帮助标识，用于查看命令用法。

## 输出说明

退出前控制台返回 `再见!` 后工具进程结束。

## 示例

非交互式方式（展示命令与回显）：

```bash
ascend-fd-tk exit
再见!
```

交互式方式（展示命令与回显）：

```bash
ascend-fd-tk
>>> exit
再见!
```
