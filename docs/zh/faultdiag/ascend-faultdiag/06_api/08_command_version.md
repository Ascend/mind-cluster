# version命令（显示版本信息）

## 功能说明

显示 ascend-fd 的版本信息。

## 命令格式

```shell
ascend-fd version [-h]
```

## 参数说明

| 参数       | 类型 | 必选 | 说明         |
|------------|------|------|--------------|
| -h, --help | -    | 否   | 显示帮助信息 |

## 使用示例

```shell
ascend-fd version
```

回显示例：

```Text
ascend-fd v26.0.0
```
