# 常用操作

## 查询 NPU 信息

查询 NPU 驱动和固件版本：

```shell
npu-smi info
```

回显示例：

```text
+--------------------------------------------------------------------------------------------------------+
| npu-smi xxx                                             Version: xxx                                   |
+-------------------------------+-----------------+------------------------------------------------------+
| NPU     Name                  | Health          | Power(W)     Temp(C)           Hugepages-Usage(page) |
| Chip    Device                | Bus-Id          | AICore(%)    Memory-Usage(MB)                        |
+===============================+=================+======================================================+
| 0       xxx                   | OK              | 10.4          66               0    / 0              |
| 0       0                     | NA              | 0            957   / 10810                           |
+===============================+=================+======================================================+
```

## 环境变量

| 环境变量              | 说明                                                                                               |
|-----------------------|----------------------------------------------------------------------------------------------------|
| `ASCEND_FD_HOME_PATH` | 运行日志、操作日志、配置文件的存放路径，默认为 `$HOME/.ascend_faultdiag`，不支持设置到 /tmp 目录下 |
