# 常用操作

## 查询 NPU 信息

查询 NPU 驱动和固件版本：

```shell
npu-smi info
```

## 环境变量

| 环境变量              | 说明                                                                                                           |
|-----------------------|----------------------------------------------------------------------------------------------------------------|
| `ASCEND_FD_HOME_PATH` | 存放运行日志，操作日志，自定义日志和配置文件存放路径，默认为 `$HOME/.ascend_faultdiag`，不支持设置 /tmp 目录下 |
