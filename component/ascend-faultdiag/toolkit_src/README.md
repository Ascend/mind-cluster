# MindCluster 链路故障诊断工具

**MindCluster Ascend FaultDiag Toolkit** 提供昇腾 AI 集群链路故障诊断能力的轻量工具，支持从 PC 或单台服务器远程访问集群设备采集数据诊断。

## 📢 变更通知

- **2026-06-05**: ✨ 诊断报告表格展示优化
- **2026-06-05**: ✨ 新增链路精准定位
- **2026-03-06**: 🚀 提供链路故障诊断分析能力
